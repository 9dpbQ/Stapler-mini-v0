## どんなアプリ？
Stapler-mini-v0は[Karabiner-Elements](https://karabiner-elements.pqrs.org/)で使えるjsonファイルを出力するmacOSアプリケーションです。
いわゆる自作キーボードにおける、レイヤー機能・タップホールド機能を、ドラッグ&ドロップで直感的にカスタマイズすることができます。
